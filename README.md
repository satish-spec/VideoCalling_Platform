Video Calling Platform

A full-stack video calling platform that enables users to create and join secure video
meetings in real time. The application provides seamless audio/video communication, unique
meeting rooms, and a responsive interface for an engaging collaboration experience.




Features
  - High-quality real-time video and audio calling
  - Create and join meetings using a unique room ID
  - Multi-user video conferencing
  - Mute and unmute microphone
  - Turn camera on/off during meetings
  - Real-time chat messaging
  - Low-latency communication using WebRTC & Socket.IO




Tech Stack
  - React.js
  - Node.js
  - Express.js
  - Socket.IO
  - WebRTC
  - HTML5
  - CSS3
  - JavaScript
